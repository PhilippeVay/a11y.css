Nouveau : le bookmarklet.
Vous pouvez l’ajouter simplement en glissant le bouton rouge vers votre barre de favoris. Vous pourrez ensuite l’actionner sur n’importe quel site !

Sur GitHub a11y.css

Prononcez « Alix », c’est rigolo et ça mange pas de pain.

L’objectif de ce fichier CSS est d’alerter l’intégrateur sur les erreurs et risques potentiels dans le code — mais il peut également servir à auditer rapidement un site existant afin d’évaluer l’ampleur des dégâts.

Accordez une seconde à ce bookmarklet pour se charger, car il va injecter un appel à a11y.css directement dans le code source de la page que vous consultez. Ça peut prendre un peu de temps puisque le fichier chargé est appellé depuis GitHub, mais ainsi il sera toujours à jour !
